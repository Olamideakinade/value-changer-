# Value Changer

A simple web app to change the case of text input:

- Upper Case-
- Lower Case
- Random Case
- Copy to Clipboard
- Reset input

## How to Use

1. Open `index.html` in a browser.
2. Type a value in the input field.
3. Click buttons to change or copy the value.
